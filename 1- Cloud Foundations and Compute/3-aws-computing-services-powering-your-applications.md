# Amazon Cloud Adoption Framework (CAF)

Is an organization that helps Businesses move to AWS smoothly and effectively

CAF helps the businesses from different perspectives: 
- business perspective
- people perspective
- governance perspective
- platform perspective
- security perspective
- operations perspective

How CAF helps ? 
- provides a clear could adoption roadmap
- identifies potential changes
- helps identify optimizations

## How Business profits from CAF ?

- cost optimization
- improve performance tracking
- enhance businesses security and compliance
- supports innovation by easing ideas development

## AWS Migration Strategies

### Rehost (Lift and shift)

Migrate as is
- no risk
- fast

### Replatform

Make few optimizations:
- move from self managed databases to using services

### Refactor

Rearchitecting the application to use full power of cloud

## AWS Migration Tools

AWS offers tools to ease the migration

### AWS Migration Hub

A central dashboard to track all migrations activities

### AWS Application Migration Service

Is perfect for rehosting (lift-and-shift)

### AWS Trusted Advisor

Gives the user recommendation on cost, security and performance before and after migration

# Moving Data to AWS

Helps moving data to the cloud securely and efficiently

## AWS Snowball

- physical device to transfer TB and GB
- data is encrypted

## AWS DMS

- supports homogeneous (Oracle to Oracle) and heterogeneous (Oracle to PostgreSql)
- real-time replication during sync => no downtime
